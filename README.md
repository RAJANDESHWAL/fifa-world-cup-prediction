# FIFA World Cup Match Prediction
- Built an end-to-end Machine Learning system to predict FIFA World Cup match outcomes (Home Win, Away Win, Draw) and total goals using historical FIFA World Cup data.
- The best-performing model was **XGBoost Classifier**, achieving **62% accuracy** on the 2022 FIFA World Cup test dataset.
- Engineered football-specific features including average goals, World Cup experience, host nation advantage, penalty shootout experience, and discipline score to improve predictive performance.
- Completed bonus tasks by implementing **GridSearchCV** for hyperparameter tuning and developing an interactive **Streamlit web application** for real-time match prediction with probability outputs.
- The project demonstrates a complete machine learning workflow including preprocessing, feature engineering, model optimization, evaluation, and deployment.
